title: 类型交集
sidebar:

  label: 15. 类型交集
---

交集类型由 `&` 运算符定义，是扩展类型的主要机制，`extends` 仅适用于 `interface` ：

```typescript
type A = {
    a: string;
};
type B = {
    b: string;
};
type C = A & B;
```

或者：

```typescript
interface X {
    x: string;
}
interface Y {
    y: string;
}
type J = X & Y;
```
